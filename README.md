# 📖 About The Project

**Code Encryption & Decryption Generator** is a Python-based application developed to explore the fundamentals of **data security**, **text encoding**, and **cryptography**.

The application allows users to convert plain text into an encoded format, encrypt the encoded data, and later decrypt it back to its original form. The project focuses on understanding how encryption algorithms work and how sensitive information can be protected during storage and transmission.

It serves as an educational project for students and beginners interested in cybersecurity and secure software development.

# ⚙️ How It Works

1. Enter plain text.
2. Convert the text into an encoded format.
3. Encrypt the encoded data using cryptographic methods.
4. Store or transmit the encrypted output securely.
5. Decrypt the encrypted data when required.
6. Recover the original text successfully.



